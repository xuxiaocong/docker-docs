# Docker 官方文档翻译

* 最近在学习Docker，网上找了找，翻译也好教程也好，都很早之前的版本，很多都已经过时了，甚至demo都跑不通；
* 直接使用谷歌翻译往往不是很准确，看不明白，只好官方英文文档硬看，速度贼慢；
* 心血来潮就试试自己翻译吧，造福下后来者；
* 使用人工翻译，过程中口语化比较多，为了符合国人阅读习惯，会删减比较老美口语化的东西，但是尽量保证表述正确；
* 排版使用中英对照，如果觉得翻译得不太对自己瞄下原文；
